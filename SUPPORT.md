# Support

Use GitHub Issues for reproducible defects, control proposals, and compatibility problems. Include the Hardening Lens version, Windows edition and build, PowerShell version, selected baseline, control ID, and redacted evidence.

Questions about organization-specific policy choices, risk acceptance, or whether a setting is safe for a workload require local security and application ownership. Hardening Lens reports effective technical state; it does not decide business risk.

Security-sensitive reports belong in GitHub Security Advisories, not public issues. See [SECURITY.md](SECURITY.md).
